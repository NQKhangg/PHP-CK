# PHP-CK
Đồ án cuối kì PHP
Dự án PHP cuối cùng
Tính năng
Xác thực và người dùng đăng nhập
Danh mục sản phẩm với phân loại
Chức năng giỏ hàng
bảo lý đơn hàng SQL
Bảng điều khiển quản trị để quản lý sản phẩm, đơn hàng và người dùng, hệ thống sản phẩm được mua trong ngày.
Yêu cầu
Kho sao chép:

git clone https:https://github.com/NQKhangg/PHP-CK.git
Điều hướng đến dự án thư mục:

cd final_php
Bảo cơ sở dữ liệu:

Nhập tệp ogtech.sqlnằm trong thư mục databasevào máy chủ MySQL của bạn.
Khởi động máy chủ:

Đặt dự án vào thư mục htdocscủa XAMPP và khởi động Apache và MySQL.
Truy cập ứng dụng:

Mở trình duyệt và truy cập http://localhost/final_php.
